-Jalankan software XAMPP , dan aktifkan Apache dan MySQL

-Download file lapor.sql

-Buat database dengan nama "lapor"

-Import file lapor.sql yang telah didwonload tadi.

-Ekstrak file.zip ke direktori C:/xampp/htdocs
